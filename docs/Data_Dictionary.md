# Data Dictionary

## Table

Review_patient_history_samp

## Description

Patient demographic information used for exploratory analysis.

| Column | Data Type | Description |
|---------|-----------|-------------|
| ID | Integer | Unique patient identifier |
| Age | Text | Patient age category |
| Gender | Text | Patient gender |
| Income | Text | Patient income category |

## Notes

- One record represents one patient.
- Age is stored as categorical ranges.
- Income is stored as categorical ranges.
- Dataset contains 5,000 patient records.
