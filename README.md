# Healthcare Patient Demographics Analysis

## Project Overview

This project demonstrates an end-to-end healthcare data analysis workflow using SQL Server.

The objective was to analyze patient demographic data, assess data quality, answer business questions, and present findings through SQL queries and visualizations.

---

## Business Questions

This analysis answers questions such as:

- How many patients are included in the dataset?
- What is the gender distribution?
- Which age groups are most represented?
- What are the income distributions?
- Are there missing demographic values?
- How do age and income distributions compare?
